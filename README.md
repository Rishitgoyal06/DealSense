# DealSense

DealSense is a web-based follow-up and deal intelligence platform designed for real estate brokers.

## Problem
Real estate brokers manage leads and follow-ups using WhatsApp and Excel, leading to missed opportunities and poor prioritization.

## Solution
DealSense centralizes leads, properties, and follow-ups in one system and provides a clear view of which leads are deal-ready.

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Cloud: AWS
